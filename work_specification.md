# 🍊 과일 주문 데이터 분석 대시보드 프로젝트 작업지시서

## 1. 프로젝트 개요
본 프로젝트는 `project1 - preprocessed_data.csv` 데이터를 활용하여 셀러별 성과, 유입 경로별 기여도, 지역별 매출 특성 및 고객 로열티를 분석하고 시각화하는 것을 목적으로 함. 특히 대형 인플루언서(킹댕즈)와 일반 셀러의 데이터를 분리하여 보다 객관적인 시장 분석을 수행함.

## 2. 주요 분석 요구사항
### 2.1 그룹 필터링
- **킹댕즈 및 일반셀러**를 구분하여 분석할 수 있는 단일/다중 선택 필터 구현.
- 각 그룹 선택 시 모든 지표와 그래프가 즉시 업데이트되어야 함.

### 2.2 탐색적 데이터 분석 (EDA) 탭 구성
- **매출 추이 분석**: 일자별 매출 및 그룹별 성과 비교.
- **채널/경로 분석**: 유입 채널별 매출 기여도 및 효율성 분석.
- **지역/연계 분석**: 광역지역별 매출과 해당 지역의 베스트 [경로x셀러] 조합 분석.
- **로열티 분석**: 재구매율이 높은 셀러 및 경로에 대한 집중 분석.

## 3. 시각화 구현 상세 (최소 요구사항)
### 3.1 시각화 라이브러리: Plotly (Interactive)
- **그래프 (5개 이상)**
    1. 선 그래프: 일자별 매출 추이
    2. 파이 차트: 주문 경로별 매출 비중
    3. 막대 그래프: 채널별 평균 객단가 (AOV)
    4. 막대 그래프: 광역지역별 총 매출액
    5. 막대 그래프: 품종별 판매량 Top 10
    6. 수평 막대 그래프: 셀러별 매출 성과 Top 20
- **표 (5개 이상)**
    1. 데이터 요약: 지역별 베스트 [경로x셀러] 조합표
    2. 셀러 요약: 매출 상위 10개 셀러 목록
    3. 로열티 요약: 재구매율 상위 10개 셀러 목록
    4. 채널 요약: 경로별 성과 지표 (매출, 주문건수, 고객수)
    5. 데이터 샘플: 최근 주문 데이터 샘플 (Raw Data)

## 4. 기술 스택
- **Python 3.x**
- **Streamlit**: 대시보드 웹 프레임워크
- **Pandas**: 데이터 전처리 및 분석
- **Plotly**: 인터랙티브 그래프 구현
- **JSON**: 연계 분석 데이터 저장 및 관리
